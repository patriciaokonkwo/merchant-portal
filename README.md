# Usage

To use the NovaPay merchant onboarding and self-service portal:

1. **Onboarding**: Merchants can sign up and complete the onboarding process by providing necessary information.
2. **KYB (Know Your Business)**: Complete the KYB verification by submitting required documents.
3. **Bank Account Linking**: Link your bank account for transactions.
4. **API Key Management**: Generate and manage your API keys for integration.

# Features

- **User-Friendly Interface**: Easy navigation for merchants.
- **Secure Document Upload**: Safe submission of sensitive documents for KYB.
- **Real-Time Status Updates**: Track the status of your onboarding and verification processes.
- **API Integration**: Seamless integration with your existing systems using provided API keys.

For more detailed instructions, refer to the specific sections in the portal.
